# NomNomGo Issue Index

This index lists GitHub-ready issue files generated from the NomNomGo planning framework. Each issue is intended to be small enough for a focused Codex task or a few days of implementation.

## Priority Key

- P0: Required for local MVP or core private-alpha safety.
- P1: Required for private alpha or public launch readiness.
- P2: Important follow-on product work.
- P3: Later strategic, monetization, business, agent, or moat work.
