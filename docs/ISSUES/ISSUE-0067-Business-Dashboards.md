# ISSUE-0067: Business Dashboards

- Title: Business Dashboards
- Epic: EPIC-009 Business Tools
- Priority: P3
- MVP: No
- Milestone: M6 - Monetization
- Story ID: BT-002

## User Story

As a venue owner, I want a dashboard so I can see planning activity.

## Description

Create a dashboard that shows business-relevant planning activity such as impressions, clicks, referrals, and locked-plan influence.

## Acceptance Criteria

- Business users can view metrics for their authorized venue or campaign.
- Dashboard distinguishes impressions, clicks, locked plans, and referrals.
- No private user or unrelated plan details are exposed.
- Empty states explain how data is generated.

## Technical Notes

- Build on analytics and attribution events.
- Keep dashboard scope narrow for first release.
