As a business, I want a venue negotiation assistant so requests can be handled efficiently.

## Description

Explore assistant workflows for business-side responses to group planning intent, offers, or reservation requests.

## Acceptance Criteria

- Business responses are scoped to authorized business accounts.
- Initial workflow is documented before implementation.

## Technical Notes
