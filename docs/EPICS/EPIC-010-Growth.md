# EPIC-010: Growth

Status: Proposed

Primary phase: Phase 1 through Phase 4

## Mission

Acquire users efficiently through planning utility.

## User Outcome

Sharing a plan should naturally bring in the right people, explain the product quickly, and convert invitees into repeat users when the planning experience works.

## Stories

| ID | Story | Priority | Acceptance Signal |
| --- | --- | --- | --- |
| GR-001 | As a user, I want an invite system so I can bring participants into a plan. | Now | Invites can be shared and opened reliably. |
| GR-002 | As a user, I want a referral program so inviting people can be rewarded when appropriate. | Later | Referral attribution is clear and abuse-resistant. |
| GR-003 | As a user, I want QR onboarding so in-person groups can join quickly. | Later | A plan or group can generate a QR join entry point. |
| GR-004 | As a participant, I want shareable plans so I can send the outcome to others. | Now | Plan links show safe, useful previews and route correctly. |
| GR-005 | As a product owner, I want viral loops so good planning moments drive distribution. | Next | Sharing, joining, voting, and locking all expose natural invite moments. |
| GR-006 | As an early user, I want waitlists so access can be managed during alpha or beta. | Later | Waitlist flows capture demand without exposing private app access. |
| GR-007 | As a new user, I want lightweight onboarding so I can join a plan before creating a heavy profile. | Now | Invitees can participate with the minimum identity needed for the plan. |

## Technical Backlog

- Invite link format.
- Invite preview metadata.
- Onboarding from invite.
- Referral tracking model.
- QR code generation.
- Waitlist model.
- Growth analytics events.
- Abuse limits for invite and referral actions.

## Definition Of Done

- A user can share a plan and a new participant can contribute without confusion.
- Growth loops are based on useful planning actions, not spam.
- Hosted alpha protection remains intact until public launch readiness.

## Metrics

- Invite sent count.
- Invite open rate.
- Invite join conversion.
- New user activation from invite.
- Referral conversion.
- Repeat use after first invited plan.
