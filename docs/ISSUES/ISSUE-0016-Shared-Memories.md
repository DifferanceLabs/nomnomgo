# ISSUE-0016: Shared Memories

- Title: Shared Memories
- Epic: EPIC-002 Social Network
- Priority: P3
- MVP: No
- Milestone: M5 - Public Launch
- Story ID: SN-006

## User Story

As a participant, I want shared memories so prior outings feel durable.

## Description

Allow participants to attach lightweight notes or memories to completed plans.

## Acceptance Criteria

- Participants can add a note to a completed plan where permissions allow.
- Notes are visible only to authorized participants or group members.
- Notes can be edited or removed by their author.
- Memory UI does not create a public feed.

## Technical Notes

- Start with text notes before adding media.
- Include moderation and deletion considerations in the model.
