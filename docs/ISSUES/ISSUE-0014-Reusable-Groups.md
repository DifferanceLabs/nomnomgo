# ISSUE-0014: Reusable Groups

- Title: Reusable Groups
- Epic: EPIC-002 Social Network
- Priority: P2
- MVP: No
- Milestone: M3 - Friends & Groups
- Story ID: SN-004

## User Story

As a user, I want groups so recurring planning circles are easy to reuse.

## Description

Add lightweight groups that can be created from repeat participants or from scratch for future planning sessions.

## Acceptance Criteria

- Users can create a group with a name and members.
- Groups can be selected when starting a new plan.
- Group membership changes are controlled by permissions.
- A group does not expose unrelated private plans.

## Technical Notes

- Define group membership roles before adding advanced settings.
- Keep the first version lightweight and planning-focused.
