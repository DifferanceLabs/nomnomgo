# NEXT

Build after MVP validation.

NEXT contains work that improves repeat behavior once the core planning loop has evidence of use.

## Product Priorities

- Friends.
- Group history.
- Shared preferences.
- Recommendation engine.
- Notifications.
- Saved places.
- Reusable plans.
- Recommendation feedback.

## Issue-Ready Outcomes

- A user can invite previous participants faster.
- A recurring group can start from prior history.
- Saved places can be reused as suggestions.
- Notifications bring participants back to plans.
- Recommendations explain why they appeared.
- Users can provide feedback on recommendations.

## Validation Signals

- 100 recurring users.
- 25 recurring groups.
- Repeat plans from the same group within 30 days.
- Recommendation selection rate is high enough to reduce external search.
