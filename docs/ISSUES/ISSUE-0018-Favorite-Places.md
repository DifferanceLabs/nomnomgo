# ISSUE-0018: Favorite Places

- Title: Favorite Places
- Epic: EPIC-002 Social Network
- Priority: P2
- MVP: No
- Milestone: M3 - Friends & Groups
- Story ID: SN-008

## User Story

As a user, I want favorite places so good options are easy to reuse.

## Description

Let users save places and reuse them as suggestions in future plans.

## Acceptance Criteria

- Users can save and unsave a place.
- Saved places can be viewed from a user profile or planning surface.
- Saved places can be added as plan suggestions.
- Provider-backed and manual places are handled consistently.

## Technical Notes

- Use the same place identity strategy as venue suggestions.
- Capture saved, unsaved, and reused events.
