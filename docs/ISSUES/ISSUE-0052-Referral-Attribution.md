# ISSUE-0052: Referral Attribution

- Title: Referral Attribution
- Epic: EPIC-007 Monetization
- Priority: P3
- MVP: No
- Milestone: M6 - Monetization
- Story ID: MN-001

## User Story

As a business owner, I want referral attribution so I know when NomNomGo influenced visits.

## Description

Track referral signals that connect planning activity to venue interest without overstating causality.

## Acceptance Criteria

- Venue clicks, locked destinations, map opens, and share events can be attributed.
- Attribution event names and properties are documented.
- Reports distinguish influenced planning activity from confirmed purchases.
- Attribution avoids collecting unnecessary personal data.

## Technical Notes

- Build on analytics taxonomy before adding monetization reporting.
- Keep attribution provider-neutral.
