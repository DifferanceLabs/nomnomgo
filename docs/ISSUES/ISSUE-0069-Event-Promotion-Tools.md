# ISSUE-0069: Event Promotion Tools

- Title: Event Promotion Tools
- Epic: EPIC-009 Business Tools
- Priority: P3
- MVP: No
- Milestone: M6 - Monetization
- Story ID: BT-004

## User Story

As a venue owner, I want event promotion so I can reach groups considering activities.

## Description

Allow businesses to promote relevant events to planning sessions through clearly labeled placements.

## Acceptance Criteria

- Businesses can create or submit event promotion details.
- Promoted events are labeled when shown to users.
- Promotions respect category, location, and timing relevance.
- Promotion performance can be tracked.

## Technical Notes

- Reuse activity recommendation and offer concepts where possible.
- Keep promoted event data distinct from provider event data.
